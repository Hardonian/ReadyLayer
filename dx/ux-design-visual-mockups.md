## Design Specifications

### Colors
- **Badge Bronze:** #CD7F32
- **Badge Silver:** #C0C0C0
- **Badge Gold:** #FFD700 (with glow effect)
- **Badge Platinum:** #E5E4E2
- **Badge Diamond:** #B9F2FF (with sparkle)

### Typography
- **Headings:** Inter, 700 weight
- **Body:** Inter, 400 weight
- **Badge Names:** Inter, 600 weight, 14px
- **Achievement Titles:** Inter, 700 weight, 24px

### Spacing
- **Card Padding:** 16px-24px
- **Component Gap:** 8px-16px
- **Section Gap:** 24px-32px

### Animations
- **Badge Unlock:** Scale + rotate animation (0.6s)
- **Achievement Celebration:** Bounce animation (0.5s)
- **Streak Fire:** Flicker animation (1s loop)
- **Page Transitions:** Fade in (0.3s)

---

## Responsive Breakpoints

### Mobile (< 640px)
- Single column layout
- Stacked cards
- Reduced font sizes
- Touch-optimized buttons

### Tablet (641px - 1024px)
- Two column layout
- Side-by-side cards
- Standard font sizes

### Desktop (> 1024px)
- Multi-column layout
- Grid-based cards
- Full feature set

---

## Accessibility

### Screen Reader Support
- ARIA labels for all badges
- Live regions for achievements
- Descriptive text for icons
- Keyboard navigation support

### Color Contrast
- WCAG AA compliance (4.5:1 minimum)
- High contrast mode support
- Color-blind friendly palettes
