# Archived Migrations

This folder contains migrations that have been successfully executed and archived.

## Archive Policy

- Migrations are automatically archived after successful execution via GitHub Actions
- Archived migrations are committed to git
- Do not manually delete archived migrations
- Archived migrations serve as a historical record

## Structure

Each archived migration maintains its original folder structure:
```
archived/
  20241230000000_init_readylayer/
    migration.sql
```
