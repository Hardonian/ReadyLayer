### Long Description
ReadyLayer brings AI code readiness to VS Code. See review feedback, generate tests, and preview documentation—all without leaving your editor.

**Features:**
- **Inline code review:** See ReadyLayer review comments directly in VS Code
- **Test generation:** Generate tests for AI-touched files on-demand
- **Documentation preview:** Preview API docs as you write code
